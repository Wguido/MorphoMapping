# Getting Started

### Dependencies
* The following Python packages are needed for MorphoMapping:
  
| Package | Version |
| --- | --- |
| `bokeh` | *V. xy*  |
| `flowkit` | *V. xy*  |
| `hdbscan` | *V. xy*  |
| `matplotlib` | *V. xy*  |
| `numpy` | *V. xy*  |
| `pandas` | *V. xy*  |
| `pandas-bokeh` | *V. xy*  |
| `seaborn` | *V. xy*  |
| `scikit-learn` | *V. xy*  |
| `umap-learn` | *V. xy*  |

* The following R libraries  are required for MorphoMapping:
  
| Library | Version |
| --- | --- |
| `IFC` | *V.0.2.1*  |
